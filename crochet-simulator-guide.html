How to Use Crochet Simulator

Welcome to this guide on using the Crochet Simulator! This online tool helps you visualize what shape you might get when following a crochet pattern. Let's learn how to use it step by step.

Getting Started

Step 1: Access the Simulator

Visit the Crochet Simulator website at: https://timhutton.github.io/crochet-simulator/

Step 2: Understanding the Interface

The interface has several parts:

  • A 3D view area where your crochet pattern will be displayed
  • An "Add stitches" input box for adding stitches one by one
  • An "Edit pattern" area for editing the entire pattern at once
  • A "Specify the row lengths" section for simplified pattern creation

Basic Usage

Step 3: Adding Your First Stitches

To start creating a pattern:

  1. Type c (for chain stitch) in the "Add stitches" box
  2. Click the "Add" button
  3. You'll see a chain stitch appear in the 3D view
  4. Repeat or type multiple characters like cccc to add several stitches at once

Tip: You can also use the "Edit pattern" box to write or paste a complete pattern and click "Update" to see it all at once.

Step 4: Understanding the Pattern Syntax

The simulator uses a specific syntax:

  • c - chain stitch
  • sc-n - single crochet joined to the stitch from n stitches ago
  • sc-n-m - single crochet joined to two stitches (useful for invisible decrease)
  • (seq)n - repeat the sequence "seq" n times

Examples:

  • (c)20 - chain 20 stitches
  • (c)20,sc-19 - make a loop of 20 chained stitches joined with a single crochet
  • c,c,c,c,sc-2,sc-4 - make two rows of 3 stitches

Advanced Features

Step 5: Crocheting in the Round

If you're crocheting in the round, you can use the "Specify the row lengths" section:

  1. Enter comma-separated numbers representing the number of stitches in each row
  2. For example: 6,12,18,24,30
  3. Click the "Go" button
  4. The simulator will compute the necessary stitches and display the result

Step 6: Controlling the View

You can interact with the 3D view:

  • Mouse up and down - moves the viewpoint up and down
  • Mouse click - starts or stops the spinning
  • Use the "Turn inflation on" button to see a more inflated version of your pattern
  • Use the "Hide target mesh" button if you're working with a target mesh

Step 7: Sharing Your Pattern

You can share your pattern with others:

  1. Create your pattern in the simulator
  2. Look for the "Share this pattern" link at the top
  3. Click on the link to get a URL that contains your pattern
  4. Share this URL with others so they can see exactly what you've created

Example Patterns

Granny Square

Try this pattern for a granny square:

(c)12(c,sc-2,sc-4,sc-6,sc-8,sc-10,sc-12,sc-14,sc-16,sc-18,sc-20,sc-22)11

Copy this into the "Edit pattern" box and click "Update" to see it.

Experimental Features

Load a Target Mesh

The simulator can help you create patterns for specific 3D shapes:

  1. Scroll down to the "Load a target mesh" section
  2. Paste in the contents of a Wavefront OBJ format mesh
  3. Click "Go"
  4. The simulator will try to create a pattern that matches that shape

Note: This feature requires that your mesh is triangular, manifold, and has approximately equal sized faces.

Tips for Success

  • Start with simple patterns to get familiar with the syntax
  • Use the examples provided to understand how different stitches work together
  • Remember that commas between stitches are optional
  • If your pattern doesn't look right, check your syntax for errors
  • For complex patterns, build them gradually to understand how each section affects the final shape

Now you're ready to use the Crochet Simulator! Experiment with different patterns and have fun seeing your creations come to life in 3D.
